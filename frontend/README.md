# TeaBoard RSS Frontend

TeaBoard RSS - 한국어 최적화 RSS 서비스 프론트엔드

## 기능

- 네오 브루탈리즘 디자인
- 한국어 사용자 친화적 인터페이스
- 반응형 웹 디자인
- GitHub 및 Vercel 배포 지원

## 기술 스택

- **Framework**: Next.js 14
- **Language**: JavaScript
- **Styling**: CSS-in-JS (styled-jsx)
- **Deployment**: Vercel

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 프로덕션 시작

```bash
npm start
```

## 배포

이 프로젝트는 Vercel에 자동 배포되도록 설정되어 있습니다.

- **Repository**: https://github.com/reallygood83/rss
- **Deployment URL**: https://question-talk.vercel.app

## 프로젝트 구조

```
frontend/
├── pages/              # Next.js 페이지
│   └── index.js       # 메인 랜딩 페이지
├── components/         # 재사용 가능한 컴포넌트
├── styles/            # 스타일 파일
├── public/            # 정적 파일
├── next.config.js     # Next.js 설정
├── package.json       # 패키지 의존성
├── tsconfig.json      # TypeScript 설정
└── vercel.json        # Vercel 배포 설정
```
